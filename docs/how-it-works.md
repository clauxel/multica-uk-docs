# How It Works - Multica Launch

Multica Launch helps teams evaluate, deploy, and operate Multica-style human plus agent teamwork with runtimes, skills, issue flow, and shared console context.

## Define team shape

List who assigns work, which agents execute it, and which runtimes must be available.

## Choose launch plan

Pick hosted or self-hosted expectations, deployment capacity, model route, and support level.

## Prepare runtime access

Verify daemon setup, CLI availability, credentials, and workspace boundaries.

## Operate repeat launches

Use the console for follow-up deployments, upgrades, and team handoff.

## Pricing and Support

Plans cover managed launch capacity, deployment guidance, console follow-up, runtime setup, and support for repeat launches.
