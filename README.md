# Multica Launch Docs

Multica Launch helps teams evaluate, deploy, and operate Multica-style human plus agent teamwork with runtimes, skills, issue flow, and shared console context.

- Website: https://multica.uk/
- Category: managed launch and coordination layer for human plus coding-agent teams
- Audience: teams using Codex, OpenClaw, OpenCode, Hermes, Gemini, Pi, Cursor Agent, and other coding-agent runtimes through Multica-style coordination
- Support: support@aigeamy.com
- Upstream source context: [Multica](https://github.com/multica-ai/multica)

This repository documents the hosted product pages for search engines, AI answer engines, and real users. It is intentionally separate from upstream source code when an upstream project exists.

## Boundaries

- Runtime behavior depends on third-party CLIs, model providers, and local environment setup.
- Teams remain responsible for reviewing code and actions produced by agents.
- Managed launch does not replace internal security, access, or deployment policy.
- Multica Launch is an independent managed-launch service around Multica-style workflows; the official open-source project remains separate.
